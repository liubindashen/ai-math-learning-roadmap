# 📚 分阶段学习资源推荐
所有资源优先选择经典、免费、易获取的内容，覆盖教材、公开课、习题三个维度，直接拿来就能用
---
## 🔴 第一阶段：筑基补漏（中小学基础）
| 模块 | 推荐资源 | 说明 |
|------|----------|------|
| 小学/初中/高中数学基础 | B站「李永乐老师高中数学合集」、《高中数学基础知识手册》 | 不用刷难题，快速过一遍知识点，把遗忘的概念捡起来即可 |

## 🟠 第二阶段：本科达标
| 模块 | 推荐资源 | 说明 |
|------|----------|------|
| 高等数学 | 教材：同济版《高等数学》第七版<br>公开课：B站「宋浩高等数学合集」<br>习题：教材课后基础题 + 《吉米多维奇高等数学习题集》（选做基础部分） | 重点掌握微积分、偏导数、梯度等和AI相关的内容，不用刷太偏的证明题 |
| 线性代数 | 教材：同济版《线性代数》第六版<br>公开课：B站「宋浩线性代数合集」/ MIT《线性代数》公开课（Strang教授）<br>习题：教材课后基础题 | MIT的公开课更侧重几何直观理解，适合想深入理解本质的学习者 |
| 概率论与数理统计 | 教材：浙大版《概率论与数理统计》第四版<br>公开课：B站「宋浩概率论与数理统计合集」<br>习题：教材课后基础题 | 重点掌握概率分布、贝叶斯定理、统计推断相关内容，和AI相关性极高 |
| 离散数学 | 教材：《离散数学及其应用》（原书第8版）<br>公开课：B站电子科技大学《离散数学》合集 | 重点掌握图论、集合论、数理逻辑部分，和AI算法、知识图谱强相关 |

## 🟡 第三阶段：硕士达标
| 模块 | 推荐资源 | 说明 |
|------|----------|------|
| 凸优化 | 教材：《凸优化》（Boyd 著）<br>公开课：Stanford CS229 凸优化部分 / Boyd 本人的凸优化公开课<br>习题：教材课后基础题 | 不用死啃复杂证明，重点理解各类优化算法的应用场景和原理 |
| 信息论 | 教材：《信息论基础》（Cover 著）<br>公开课：B站中科大《信息论》合集 | 重点掌握信息熵、交叉熵、KL散度等和大模型损失函数、评估相关的内容 |
| 数值计算 | 教材：《数值计算方法》（丁丽娟 著）<br>公开课：B站清华大学《数值分析》合集 | 重点掌握数值误差、矩阵计算稳定性相关内容，对大模型精度测试很有帮助 |
| AI专项（对抗安全/测试方向） | 公开课：Stanford CS231n 计算机视觉 / CS224n 自然语言处理 / 清华《AI安全》公开课 | 结合AI算法理解数学知识的实际应用，事半功倍 |

## 🟢 第四阶段：科研入门（博士前置）
| 模块 | 推荐资源 | 说明 |
|------|----------|------|
| 高阶优化 | 教材：《非线性规划》（Bertsekas 著）<br>公开课：Stanford CS230 深度学习进阶优化部分 | 重点理解非凸优化、随机优化在大模型训练中的应用 |
| 高阶概率 | 教材：《贝叶斯推断》（Andrew Gelman 著）<br>公开课：MIT《概率图模型》公开课 | 适合研究大模型对齐、不确定性评估方向 |
| 应用数学 | 教材：《李群李代数及其在 robotics 中的应用》、《流形学习导论》 | 适合做3D视觉、具身智能、多模态表征研究 |

## 🟣 第六阶段：领域专家级
这个阶段没有固定教材，主要靠前沿论文研读、行业技术交流、实际项目打磨，推荐关注：
- DeepSeek、OpenAI、Anthropic等头部公司的核心研发团队公开的技术分享
- 数学和AI交叉领域的前沿论文（如范畴论、拓扑学在通用人工智能中的应用）
- 顶级学术会议的tutorial和keynote分享，把握领域发展方向
