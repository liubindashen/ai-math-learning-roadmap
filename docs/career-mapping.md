# 👨💻 AI岗位能力匹配对照表
明确不同AI岗位需要掌握的数学阶段和重点模块，避免学无用内容，按需学习效率更高
---
| 岗位名称 | 需要达到的学习阶段 | 重点掌握模块 | 能力要求说明 |
|----------|-------------------|--------------|--------------|
| AI产品经理 | 第二阶段（本科达标） | 概率论、统计基础、简单优化原理 | 能看懂算法指标、理解算法能力边界，不用深入推导 |
| 大模型测试工程师 | 第三阶段（硕士达标） | 概率论与数理统计、信息论、数值计算、对抗优化基础 | 能理解大模型各类评估指标含义、定位精度误差根因、设计鲁棒性测试方案 |
| 应用算法工程师 | 第三阶段（硕士达标） | 线性代数、概率论、优化理论、信息论 | 能调优算法、理解主流模型原理、解决业务落地问题 |
| AI安全研究员 | 第四阶段（科研入门） | 概率论、信息论、对抗优化、博弈论、统计分析 | 能研究对抗样本生成、大模型安全漏洞、设计防御方案 |
| 大模型算法工程师 | 第五阶段（博士级） | 高等概率、凸优化、深度学习理论、信息论进阶 | 能做模型结构创新、预训练/微调/对齐算法优化、核心性能提升 |
| 大模型核心研发负责人/资深研究员 | 第六阶段（领域专家级） | 深度学习底层理论、泛函分析、大模型全流程数学理论、前沿交叉数学 | 能主导千亿级大模型的核心技术路线设计、解决行业重大公开问题 |
---
## 🎯 不同阶段对应工作能力参考
| 学习阶段 | 能胜任的工作复杂度 |
|----------|-------------------|
| 第二阶段（本科） | 常规大模型功能测试、简单指标统计、算法产品需求跟进 |
| 第三阶段（硕士） | 大模型性能/精度/鲁棒性测试、指标体系设计、问题根因定位、常规算法落地 |
| 第四阶段（博士前置） | 大模型专项测试（如安全对齐测试）、算法改进、简单科研性质的问题研究 |
| 第五阶段（博士） | 原创算法研发、大模型核心模块优化、顶会级科研成果产出 |
| 第六阶段（专家） | 引领技术方向、主导重大技术突破、解决行业级难题 |
